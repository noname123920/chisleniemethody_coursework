\section*{Введение}
\addcontentsline{toc}{section}{Введение}

Многие физические, технические и экономические процессы описываются дифференциальными уравнениями. Особый класс составляют \textbf{двухточечные краевые задачи} для обыкновенных дифференциальных уравнений (ОДУ) второго порядка. Такие задачи возникают, когда известны условия не в одной точке, а на двух концах некоторого отрезка (отсюда название «двухточечные»).

Простейший практический пример — распределение температуры в стержне, концы которого поддерживаются при заданных температурах. Эту задачу начал изучать французский математик Ж. Фурье ещё в начале XIX века. Если стержень изготовлен из неоднородного материала (теплопроводность меняется вдоль длины) или внутри него есть источники тепла, то математическая модель принимает вид:

\begin{equation*}
    -\frac{d}{dx}\left(k(x)\frac{du}{dx}\right) + q(x)u = f(x), \quad a < x < b
\end{equation*}

Здесь $u(x)$ — искомая функция (например, температура), $k(x)$ — коэффициент теплопроводности (может зависеть от координаты), $q(x)$ — коэффициент, описывающий теплообмен с окружающей средой (например, через боковую поверхность стержня), $f(x)$ — плотность внутренних источников тепла (например, нагревательный элемент внутри). Все эти коэффициенты предполагаются зависящими от координаты $x$, что соответствует реальным неоднородным средам.

На границах отрезка $[a, b]$ задаются дополнительные условия. В зависимости от физической постановки это могут быть:

\begin{itemize}
    \item \textbf{условие первого рода} (или условие Дирихле) — задано значение самой функции на границе (например, конец стержня поддерживается при фиксированной температуре);
    \item \textbf{условие второго рода} (условие Неймана) — задано значение производной (например, теплоизолированный конец, через который нет теплового потока);
    \item \textbf{условие третьего рода} — задана линейная комбинация функции и её производной (например, теплообмен с окружающей средой по закону Ньютона).
\end{itemize}

Аналитическое (точное) решение такой задачи удаётся найти лишь для простейших случаев. Для реальных задач с переменными коэффициентами $k(x)$, $q(x)$, $f(x)$ приходится использовать \textbf{численные методы}. Один из самых распространённых — \textbf{метод конечных разностей (МКР)}. Идея метода в том, чтобы заменить непрерывную область $[a, b]$ набором отдельных точек (узлов), а производные — приближёнными формулами через значения функции в этих точках. В результате дифференциальное уравнение превращается в систему линейных алгебраических уравнений, которую легко решить на компьютере.

\textbf{Цель данной работы} — описать метод конечных разностей для решения линейной двухточечной краевой задачи, реализовать его в среде Octave (свободный аналог MATLAB), провести вычислительные эксперименты с разными граничными условиями и оценить точность полученного решения.
