\section*{Введение}
\addcontentsline{toc}{section}{Введение}

Многие физические, технические и экономические процессы описываются дифференциальными уравнениями. Особый класс составляют \textbf{двухточечные краевые задачи} для обыкновенных дифференциальных уравнений (ОДУ) второго порядка. Такие задачи возникают, когда известны условия не в одной точке, а на двух концах некоторого отрезка — отсюда название «двухточечные».

Математическая постановка задачи, рассматриваемой в данной работе, имеет вид:
\begin{equation*}
    -\bigl(k(x)\,u'\bigr)' + q(x)\,u = f(x), \quad a < x < b,
\end{equation*}
где $u(x)$ — искомая функция, $k(x)$, $q(x)$, $f(x)$ — заданные непрерывные функции, причём $k(x) > 0$ на всём отрезке $[a, b]$. Для единственности решения на границах отрезка $[a, b]$ задаются дополнительные условия — первого, второго или третьего рода.

Аналитическое решение такой задачи удаётся найти лишь для простейших случаев. Для реальных задач с переменными коэффициентами приходится использовать \textbf{численные методы}. Один из самых распространённых — \textbf{метод конечных разностей} (МКР). Суть метода заключается в замене непрерывной области $[a, b]$ набором отдельных точек (узлов), а производных — приближёнными формулами через значения функции в этих точках. В результате дифференциальное уравнение превращается в систему линейных алгебраических уравнений (СЛАУ), которую можно решить на компьютере.

\textbf{Цель работы:} программно реализовать метод конечных разностей для решения линейной двухточечной краевой задачи для ОДУ второго порядка с переменными коэффициентами.

Для достижения поставленной цели решаются следующие \textbf{задачи}:
\begin{enumerate}
    \item Описать постановку линейной двухточечной краевой задачи для ОДУ второго порядка с разными граничными условиями.
    \item Указать условия разрешимости краевой задачи.
    \item Изложить метод конечных разностей для решения этой задачи.
    \item Реализовать в среде Octave алгоритм решения краевой задачи методом конечных разностей для одномерного стационарного уравнения теплопроводности
    \begin{equation*}
        -\frac{d}{dx}\left(k(x)\frac{du}{dx}\right) + q(x)u = f(x), \quad a < x < b,
    \end{equation*}
    где $k(x) \neq \text{const}$, $q(x) \neq \text{const}$, $f(x) \neq \text{const}$ — непрерывные функции. Требуемая точность — $10^{-4}$.
    \item Оценить погрешность численного решения по правилу Рунге.
    \item Провести вычислительные эксперименты с различными краевыми условиями.
    \item Построить графики полученного решения и график погрешности.
\end{enumerate}

\section{Постановка краевой задачи с различными граничными условиями}
Рассматривается линейная двухточечная краевая задача для ОДУ второго порядка:
\begin{equation}
    -\frac{d}{dx}\left(k(x)\frac{du}{dx}\right) + q(x)u = f(x), \quad a < x < b,
    \label{eq:ode_main}
\end{equation}
где $u(x)$ --- искомая функция, $k(x)$, $q(x)$, $f(x)$ --- заданные непрерывные на $[a,b]$ функции, причём $k(x) > 0$.

Для однозначного определения решения на концах отрезка задаются граничные условия трёх возможных типов:

\begin{itemize}
    \item \textbf{Первый род (Дирихле):} фиксированы значения функции
    \begin{equation}
        u(a) = A, \quad u(b) = B.
    \end{equation}
    \item \textbf{Второй род (Неймана):} фиксированы значения производной
    \begin{equation}
        u'(a) = \gamma, \quad u'(b) = \delta.
    \end{equation}
    \item \textbf{Третий род (Робина):} задана линейная комбинация функции и производной
    \begin{equation}
        \alpha_0 u(a) + \alpha_1 u'(a) = \gamma_a, \quad \beta_0 u(b) + \beta_1 u'(b) = \gamma_b,
    \end{equation}
    где $\alpha_0 \neq 0$, $\alpha_1 \neq 0$, $\beta_0 \neq 0$, $\beta_1 \neq 0$.
\end{itemize}

В дальнейшем для всех трёх типов граничных условий применяется одинаковый алгоритм дискретизации методом конечных разностей.

\newpage
\section{Условия разрешимости и единственности}

Для каждой из рассматриваемых краевых задач укажем условия, гарантирующие существование и единственность решения.

\textbf{Задача Дирихле.} Согласно теореме~15.1 из~\cite{amosov_dubinsky_kopchenova_1994}, решение краевой задачи с условиями первого рода существует и единственно при выполнении условий:
\[
k(x) \ge k_0 > 0, \quad q(x) \ge 0, \quad k \in C^1[a,b], \quad q,f \in C[a,b],
\]
где $k_0 > 0$ — некоторая постоянная (нижняя граница функции $k(x)$ на $[a,b]$), которая выбирается исходя из минимального значения $k(x)$ на отрезке и гарантирует, что $k(x)$ отделена от нуля.
В тестовом примере $k(x) = x^2 + 1 \ge 1 > 0$, $q(x) = x + 1 \ge 1 > 0$, все функции непрерывны. Условия теоремы выполнены, решение существует и единственно.

\textbf{Задачи Неймана и Робина.} Для задач с граничными условиями второго и третьего рода существование и единственность решения устанавливаются с помощью \textbf{теоремы~3.2.2} из~\cite{denisov_razgulin_2009}. Теорема утверждает: если однородная краевая задача
\[
-\frac{d}{dx}\left(k(x)\frac{du}{dx}\right) + q(x)u = 0, \quad a < x < b,
\]
с однородными граничными условиями (для Неймана: $u'(a)=0$, $u'(b)=0$; для Робина: $\alpha_0 u(a)+\alpha_1 u'(a)=0$, $\beta_0 u(b)+\beta_1 u'(b)=0$) имеет только нулевое решение $u \equiv 0$, то решение исходной неоднородной задачи существует и единственно.

Проверка условия теоремы (доказательство того, что данная однородная задача с однородными граничными условиями имеет только нулевое решение) относится к теории краевых задач для обыкновенных дифференциальных уравнений, которая не входит в программу курса численных методов. В настоящей работе данная проверка не приводится. В тестовых примерах $k(x) = x^2 + 1 \ge 1 > 0$, $q(x) > 0$ на $[0,1]$, а коэффициенты граничных условий положительны. Сходимость численного решения для всех трёх задач с порядком $O(h^2)$ косвенно подтверждает корректность постановки.

\section{Вывод разностных формул}

В этом разделе подробно, шаг за шагом, показано, как из непрерывной 
краевой задачи получаются коэффициенты матрицы СЛАУ. Для построения 
консервативной разностной схемы используется 
\textbf{интегро-интерполяционный метод (метод баланса)}~\cite{balans_method}.

\subsection{Интегро-интерполяционный метод (метод баланса)}

Рассматривается уравнение
\begin{equation}
    -\bigl(k(x)u'(x)\bigr)' + q(x)u(x) = f(x), \quad a < x < b.
\end{equation}

Введём равномерную сетку:
\[
x_i = a + (i-1)h, \quad i = 1, 2, \dots, N, \qquad h = \frac{b-a}{N-1}.
\]
Обозначим: $u_i = u(x_i)$, $q_i = q(x_i)$, $f_i = f(x_i)$.

Введём поток \(W(x) = -k(x)u'(x)\). Тогда уравнение переписывается как
\begin{equation}
    -W' - q u = -f \quad\Rightarrow\quad W' + q u = f.
\end{equation}

Проинтегрируем последнее на отрезке \([x_{i-1/2}, x_{i+1/2}]\), где 
\(x_{i\pm1/2} = x_i \pm h/2\):
\begin{equation}
    \int_{x_{i-1/2}}^{x_{i+1/2}} W' dx + \int_{x_{i-1/2}}^{x_{i+1/2}} q u \, dx = \int_{x_{i-1/2}}^{x_{i+1/2}} f \, dx.
\end{equation}

Первый интеграл вычисляется точно:
\begin{equation}
    W(x_{i+1/2}) - W(x_{i-1/2}) + \int_{x_{i-1/2}}^{x_{i+1/2}} (q u - f) dx = 0.
\end{equation}

Это уравнение баланса на элементарной ячейке.

\subsection{Аппроксимация интегралов и потоков}

Заменим интегралы по формуле прямоугольников, полагая подынтегральные функции 
постоянными на интервале интегрирования:
\begin{equation}
    \int_{x_{i-1/2}}^{x_{i+1/2}} q u \, dx \approx q_i u_i h, \qquad
    \int_{x_{i-1/2}}^{x_{i+1/2}} f \, dx \approx f_i h.
\end{equation}

Для аппроксимации потоков \(W_{i-1/2} = W(x_{i-1/2})\) и \(W_{i+1/2} = W(x_{i+1/2})\) 
используем определение \(W = -k u'\). Проинтегрируем равенство \(u' = -W/k\) 
на отрезке \([x_{i-1}, x_i]\):
\begin{equation}
    u_{i-1} - u_i = \int_{x_{i-1}}^{x_i} \frac{W}{k} dx.
\end{equation}

Считая поток \(W = W_{i-1/2}\) постоянным на этом интервале, получаем:
\begin{equation}
    u_{i-1} - u_i \approx W_{i-1/2} \int_{x_{i-1}}^{x_i} \frac{dx}{k(x)}.
\end{equation}

Отсюда выражаем поток:
\begin{equation}
    W_{i-1/2} \approx -a_i \frac{u_i - u_{i-1}}{h}, \qquad
    a_i = \left( \frac{1}{h} \int_{x_{i-1}}^{x_i} \frac{dx}{k(x)} \right)^{-1}.
\end{equation}

Аналогично для правой границы ячейки:
\begin{equation}
    W_{i+1/2} \approx -a_{i+1} \frac{u_{i+1} - u_i}{h}, \qquad
    a_{i+1} = \left( \frac{1}{h} \int_{x_i}^{x_{i+1}} \frac{dx}{k(x)} \right)^{-1}.
\end{equation}

Подставляя последние формулы в уравнение баланса, получаем:
\begin{equation}
    -a_{i+1} \frac{u_{i+1} - u_i}{h} + a_i \frac{u_i - u_{i-1}}{h} + (q_i u_i - f_i) h = 0.
\end{equation}

Разделим на \(h\):
\begin{equation}
    \frac{1}{h}\left( -a_{i+1} \frac{u_{i+1} - u_i}{h} + a_i \frac{u_i - u_{i-1}}{h} \right) + q_i u_i = f_i, \quad i = 2, \dots, N-1.
\end{equation}

Если коэффициенты \(k(x)\) являются гладкими функциями, то с точностью до \(O(h^2)\) 
можно положить:
\[
a_i = k(x_i - h/2) \equiv k_{i-1/2}, \qquad a_{i+1} = k(x_i + h/2) \equiv k_{i+1/2}.
\]

Подставим эти выражения в уравнение:
\[
\frac{1}{h}\left( -k_{i+1/2} \frac{u_{i+1} - u_i}{h} + k_{i-1/2} \frac{u_i - u_{i-1}}{h} \right) + q_i u_i = f_i.
\]

Раскроем скобки:
\[
\frac{-k_{i+1/2}(u_{i+1} - u_i) + k_{i-1/2}(u_i - u_{i-1})}{h^2} + q_i u_i = f_i.
\]

Раскроем скобки в числителе:
\[
\frac{-k_{i+1/2}u_{i+1} + k_{i+1/2}u_i + k_{i-1/2}u_i - k_{i-1/2}u_{i-1}}{h^2} + q_i u_i = f_i.
\]

Сгруппируем слагаемые при \(u_{i-1}\), \(u_i\) и \(u_{i+1}\):
\begin{equation}
    -\frac{k_{i-1/2}}{h^2} u_{i-1} + 
    \left( \frac{k_{i-1/2} + k_{i+1/2}}{h^2} + q_i \right) u_i - 
    \frac{k_{i+1/2}}{h^2} u_{i+1} = f_i.
    \label{eq:internal_node}
\end{equation}

Это основное разностное уравнение для внутренних узлов.

В программе оно реализовано следующим циклом:

\begin{verbatim}
    for i = 2:N-1
        a_diag(i-1) = -kmid(i-1)/h^2;
        c_diag(i)   = -kmid(i)/h^2;
        b_diag(i)   = (kmid(i-1)+kmid(i))/h^2 + qv(i);
        d_vec(i)    = fv(i);
    end
\end{verbatim}

\subsection{Аппроксимация граничных условий методом фиктивных узлов}

Для сохранения второго порядка аппроксимации при учёте
граничных условий второго и третьего рода используется
метод фиктивных узлов (законтурных точек), подробно
описанный в \cite{marchuk_methods}.
Сеточная область продолжается за пределы отрезка $[a,b]$:
вводятся дополнительные точки
\[
x_0 = a - h, \qquad x_{N+1} = b + h,
\]
со значениями искомой функции $u_0 = u(x_0)$ и $u_{N+1} = u(x_{N+1})$.
Производные на границах аппроксимируются центральными
разностями, что обеспечивает второй порядок точности:
\begin{equation}
    u'(a) \approx \frac{u_2 - u_0}{2h}, \qquad
    u'(b) \approx \frac{u_{N+1} - u_{N-1}}{2h}.
    \label{eq:neumann_approx}
\end{equation}
Далее фиктивные значения $u_0$ и $u_{N+1}$ исключаются
путём выражения их из соответствующих граничных условий
и подстановки в разностные уравнения для крайних узлов
$i=1$ и $i=N$.

\subsubsection{Левая граница: условие Неймана}

Пусть задано \(u'(a) = \gamma\). Из формулы \eqref{eq:neumann_approx} 
для производной выразим \(u_0\):
\[
\frac{u_2 - u_0}{2h} = \gamma \quad\Rightarrow\quad u_0 = u_2 - 2h\gamma.
\]

Запишем уравнение \eqref{eq:internal_node} для левого граничного узла \(i=1\). 
При этом \(k_{i-1/2} = k(a - h/2) \equiv k_{-1/2}\), 
\(k_{i+1/2} = k(a + h/2) \equiv k_{1/2}\):
\[
-\frac{k_{-1/2}}{h^2}u_0 + 
\left( \frac{k_{-1/2}+k_{1/2}}{h^2} + q_1 \right)u_1 - 
\frac{k_{1/2}}{h^2}u_2 = f_1.
\]

Подставляем выражение для \(u_0\):
\[
-\frac{k_{-1/2}}{h^2}(u_2 - 2h\gamma) + 
\left( \frac{k_{-1/2}+k_{1/2}}{h^2} + q_1 \right)u_1 - 
\frac{k_{1/2}}{h^2}u_2 = f_1.
\]

Раскрываем скобки и группируем слагаемые:
\[
-\frac{k_{-1/2}+k_{1/2}}{h^2}u_2 + 
\left( \frac{k_{-1/2}+k_{1/2}}{h^2} + q_1 \right)u_1 + 
\frac{2k_{-1/2}}{h}\gamma = f_1.
\]

Переносим известное слагаемое с \(\gamma\) в правую часть:
\begin{equation}
    \left( \frac{k_{-1/2}+k_{1/2}}{h^2} + q_1 \right)u_1 - 
    \frac{k_{-1/2}+k_{1/2}}{h^2}u_2 = f_1 - \frac{2k_{-1/2}}{h}\gamma.
    \label{eq:left_neumann_final}
\end{equation}

В программе это соответствует:

\begin{verbatim}
    k_minus_half = k(a - h/2);
    k_plus_half  = k(a + h/2);
    b_diag(1) = (k_minus_half + k_plus_half)/h^2 + qv(1);
    c_diag(1) = -(k_minus_half + k_plus_half)/h^2;
    d_vec(1)  = fv(1) - 2*k_minus_half/h * gamma;
\end{verbatim}

\subsubsection{Левая граница: условие Робина}

Пусть задано \(\alpha_0 u(a) + \alpha_1 u'(a) = \gamma_a\). 
Подставляем аппроксимацию производной:
\[
\alpha_0 u_1 + \alpha_1 \cdot \frac{u_2 - u_0}{2h} = \gamma_a.
\]

Выражаем \(u_0\):
\[
\alpha_1 \cdot \frac{u_2 - u_0}{2h} = \gamma_a - \alpha_0 u_1
\quad\Rightarrow\quad
u_2 - u_0 = \frac{2h}{\alpha_1}(\gamma_a - \alpha_0 u_1)
\quad\Rightarrow\quad
u_0 = u_2 - \frac{2h}{\alpha_1}(\gamma_a - \alpha_0 u_1).
\]

Подставляем в уравнение для \(i=1\):
\[
-\frac{k_{-1/2}}{h^2}\left[ u_2 - \frac{2h}{\alpha_1}(\gamma_a - \alpha_0 u_1) \right] + 
\left( \frac{k_{-1/2}+k_{1/2}}{h^2} + q_1 \right)u_1 - 
\frac{k_{1/2}}{h^2}u_2 = f_1.
\]

Раскрываем скобки и приводим подобные:
\[
-\frac{k_{-1/2}}{h^2}u_2 + \frac{2k_{-1/2}}{h\alpha_1}\gamma_a - 
\frac{2k_{-1/2}\alpha_0}{h\alpha_1}u_1 + 
\left( \frac{k_{-1/2}+k_{1/2}}{h^2} + q_1 \right)u_1 - 
\frac{k_{1/2}}{h^2}u_2 = f_1.
\]

Объединяем слагаемые при \(u_1\) и \(u_2\):
\[
-\frac{k_{-1/2}+k_{1/2}}{h^2}u_2 + 
\left( \frac{k_{-1/2}+k_{1/2}}{h^2} + q_1 - \frac{2k_{-1/2}\alpha_0}{h\alpha_1} \right)u_1 + 
\frac{2k_{-1/2}}{h\alpha_1}\gamma_a = f_1.
\]

Переносим известное слагаемое в правую часть:
\begin{equation}
    \left( \frac{k_{-1/2}+k_{1/2}}{h^2} + q_1 - \frac{2k_{-1/2}\alpha_0}{h\alpha_1} \right)u_1 - 
    \frac{k_{-1/2}+k_{1/2}}{h^2}u_2 = f_1 - \frac{2k_{-1/2}}{h\alpha_1}\gamma_a.
    \label{eq:left_robin_final}
\end{equation}

В программе это соответствует:

\begin{verbatim}
    k_minus_half = k(a - h/2);
    k_plus_half  = k(a + h/2);
    b_diag(1) = (k_minus_half + k_plus_half)/h^2 + qv(1) - 2*k_minus_half*alpha0/(h*alpha1);
    c_diag(1) = -(k_minus_half + k_plus_half)/h^2;
    d_vec(1)  = fv(1) - 2*k_minus_half/(h*alpha1)*gamma_a;
\end{verbatim}

\subsubsection{Правая граница: условие Неймана}

Для правой границы \(i=N\) с условием \(u'(b) = \delta\) выражаем \(u_{N+1}\):
\[
\frac{u_{N+1} - u_{N-1}}{2h} = \delta \quad\Rightarrow\quad u_{N+1} = u_{N-1} + 2h\delta.
\]

Записываем уравнение \eqref{eq:internal_node} для \(i=N\), где 
\(k_{N-1/2} = k(b - h/2)\), \(k_{N+1/2} = k(b + h/2)\):
\[
-\frac{k_{N-1/2}}{h^2}u_{N-1} + 
\left( \frac{k_{N-1/2}+k_{N+1/2}}{h^2} + q_N \right)u_N - 
\frac{k_{N+1/2}}{h^2}u_{N+1} = f_N.
\]

Подставляем выражение для \(u_{N+1}\):
\[
-\frac{k_{N-1/2}}{h^2}u_{N-1} + 
\left( \frac{k_{N-1/2}+k_{N+1/2}}{h^2} + q_N \right)u_N - 
\frac{k_{N+1/2}}{h^2}(u_{N-1} + 2h\delta) = f_N.
\]

Раскрываем скобки и группируем:
\[
-\frac{k_{N-1/2}+k_{N+1/2}}{h^2}u_{N-1} + 
\left( \frac{k_{N-1/2}+k_{N+1/2}}{h^2} + q_N \right)u_N - 
\frac{2k_{N+1/2}}{h}\delta = f_N.
\]

Переносим известное слагаемое в правую часть:
\begin{equation}
    -\frac{k_{N-1/2}+k_{N+1/2}}{h^2}u_{N-1} + 
    \left( \frac{k_{N-1/2}+k_{N+1/2}}{h^2} + q_N \right)u_N = 
    f_N + \frac{2k_{N+1/2}}{h}\delta.
    \label{eq:right_neumann_final}
\end{equation}

В программе это соответствует:

\begin{verbatim}
    k_minus_half = k(b - h/2);
    k_plus_half  = k(b + h/2);
    a_diag(N-1) = -(k_minus_half + k_plus_half)/h^2;
    b_diag(N)   = (k_minus_half + k_plus_half)/h^2 + qv(N);
    d_vec(N)    = fv(N) + 2*k_plus_half/h * delta;
\end{verbatim}

\subsubsection{Правая граница: условие Робина}

Для условия \(\beta_0 u(b) + \beta_1 u'(b) = \gamma_b\) выражаем \(u_{N+1}\):
\[
\beta_0 u_N + \beta_1 \cdot \frac{u_{N+1} - u_{N-1}}{2h} = \gamma_b
\quad\Rightarrow\quad
u_{N+1} = u_{N-1} + \frac{2h}{\beta_1}(\gamma_b - \beta_0 u_N).
\]

Подставляем в уравнение для \(i=N\) и выполняем преобразования, аналогичные случаю левой границы:
\begin{equation}
    -\frac{k_{N-1/2}+k_{N+1/2}}{h^2}u_{N-1} + 
    \left( \frac{k_{N-1/2}+k_{N+1/2}}{h^2} + q_N + \frac{2k_{N+1/2}\beta_0}{h\beta_1} \right)u_N = 
    f_N + \frac{2k_{N+1/2}}{h\beta_1}\gamma_b.
    \label{eq:right_robin_final}
\end{equation}

В программе это соответствует:

\begin{verbatim}
    k_minus_half = k(b - h/2);
    k_plus_half  = k(b + h/2);
    a_diag(N-1) = -(k_minus_half + k_plus_half)/h^2;
    b_diag(N)   = (k_minus_half + k_plus_half)/h^2 + qv(N) + 2*k_plus_half*beta0/(h*beta1);
    d_vec(N)    = fv(N) + 2*k_plus_half/(h*beta1)*gamma_b;
\end{verbatim}

\subsection{Условия Дирихле}

Для условий первого рода граничные значения известны: \(u_1 = A\), \(u_N = B\). 
Поэтому первое и последнее уравнения системы заменяются на тождества:
\[
u_1 = A, \qquad u_N = B.
\]

В программе это реализовано так:

\begin{verbatim}
    b_diag(1) = 1; c_diag(1) = 0; d_vec(1) = A;
    a_diag(N-1) = 0; b_diag(N) = 1; d_vec(N) = B;
\end{verbatim}

\subsection{Итоговая система}

После описанных преобразований получается система линейных алгебраических 
уравнений с трёхдиагональной матрицей, которая эффективно решается 
методом прогонки.

\newpage
\section{Метод прогонки решения трёхдиагональной СЛАУ}

Алгоритм решения системы с трёхдиагональной матрицей, определяемый 
формулами прямого и обратного хода, называется методом прогонки 
и подробно изложен в~\cite{samarskii_gulin_1989}.

В результате дискретизации получена система линейных алгебраических уравнений 
с трёхдиагональной матрицей:
\[
\begin{cases}
b_1 u_1 + c_1 u_2 = d_1, \\
a_2 u_1 + b_2 u_2 + c_2 u_3 = d_2, \\
\cdots \\
a_{i} u_{i-1} + b_i u_i + c_i u_{i+1} = d_i, \quad i = 2, 3, \dots, N-1, \\
\cdots \\
a_{N} u_{N-1} + b_N u_N = d_N,
\end{cases}
\]
где \(a_i\), \(b_i\), \(c_i\) — коэффициенты, \(d_i\) — правая часть.

Для решения такой системы используется \textbf{метод прогонки} — 
частный случай метода Гаусса, учитывающий трёхдиагональную структуру 
матрицы и имеющий вычислительную сложность \(O(N)\).

\subsection{Алгоритм метода прогонки}

Прямой ход прогонки (вычисление прогоночных коэффициентов \(\alpha_i\) и \(\beta_i\)):
\[
\alpha_1 = \frac{c_1}{b_1}, \quad \beta_1 = \frac{d_1}{b_1},
\]
\[
\alpha_i = \frac{c_i}{b_i - a_i \alpha_{i-1}}, \quad
\beta_i = \frac{d_i - a_i \beta_{i-1}}{b_i - a_i \alpha_{i-1}}, \quad i = 2, 3, \dots, N-1.
\]

Обратный ход (восстановление решения):
\[
u_N = \frac{d_N - a_N \beta_{N-1}}{b_N - a_N \alpha_{N-1}},
\]
\[
u_i = \beta_i - \alpha_i u_{i+1}, \quad i = N-1, N-2, \dots, 1.
\]

Условия корректности метода (достаточное условие устойчивости):
\[
|b_i| \ge |a_i| + |c_i|, \quad i = 1, \dots, N,
\]
причём хотя бы для одного \(i\) неравенство строгое. 
Для рассматриваемой краевой задачи эти условия выполняются, 
так как \(k_{i\pm1/2} > 0\) и \(q_i \ge 0\).

\subsection{Программная реализация в Octave}

Ниже приведён фрагмент кода, реализующий метод прогонки для 
трёхдиагональной системы.

\begin{verbatim}
n = length(d_vec);
aa = a_diag(1:n-1);
bb = b_diag;
cc = c_diag(1:n-1);
dd = d_vec;

alpha = zeros(n, 1);
beta  = zeros(n, 1);

alpha(1) = cc(1) / bb(1);
beta(1)  = dd(1) / bb(1);

for i = 2:n-1
    denominator = bb(i) - aa(i-1) * alpha(i-1);
    alpha(i) = cc(i) / denominator;
    beta(i)  = (dd(i) - aa(i-1) * beta(i-1)) / denominator;
end

U = zeros(n, 1);
U(n) = (dd(n) - aa(n-1) * beta(n-1)) / (bb(n) - aa(n-1) * alpha(n-1));

for i = n-1:-1:1
    U(i) = beta(i) - alpha(i) * U(i+1);
end
\end{verbatim}

\subsection*{Приложение 1. Код программы для задачи Дирихле}
\label{app:dirichlet}

\begin{verbatim}
graphics_toolkit("qt");

% Определяем границы интервала
a = 0; b = 1;

% Коэффициенты дифференциального уравнения
k = @(x) x.^2 + 1;
q = @(x) x + 1;
f = @(x) x.^5 - 21*x.^4 + 23*x.^3 - 14*x.^2 + 9*x - 2;

% Точное решение и граничные условия
u_exact = @(x) x.^4 - 2*x.^3 + x.^2 + 3*x;
A = u_exact(a);
B = u_exact(b);

% Параметры метода
tol = 1e-4;      % Требуемая точность
N = 5;           % Начальное число узлов
err_est = tol + 1;
first = true;

% Основной цикл автоматического сгущения сетки
while err_est > tol
    h = (b - a)/(N - 1);          % Шаг сетки
    x = linspace(a, b, N)';       % Узлы сетки
    xmid = (x(1:end-1) + x(2:end))/2;  % Середины отрезков

    kmid = k(xmid);
    qv = q(x);
    fv = f(x);

    % Заполнение трехдиагональной матрицы
    a_diag = zeros(N,1);  % Нижняя диагональ
    b_diag = zeros(N,1);  % Главная диагональ
    c_diag = zeros(N,1);  % Верхняя диагональ
    d_vec = zeros(N,1);   % Вектор правой части

    % Внутренние узлы
    for i = 2:N-1
        a_diag(i-1) = -kmid(i-1)/h^2;
        c_diag(i)   = -kmid(i)/h^2;
        b_diag(i)   = (kmid(i-1) + kmid(i))/h^2 + qv(i);
        d_vec(i)    = fv(i);
    end

    % Граничные условия (Дирихле)
    b_diag(1) = 1; c_diag(1) = 0; d_vec(1) = A;
    a_diag(N-1) = 0; b_diag(N) = 1; d_vec(N) = B;

    % Прогонка (решение трехдиагональной системы)
    n = length(d_vec);
    aa = a_diag(1:n-1);
    bb = b_diag;
    cc = c_diag(1:n-1);
    dd = d_vec;

    % Прямой ход прогонки
    for i = 2:n
        w = aa(i-1) / bb(i-1);
        bb(i) = bb(i) - w * cc(i-1);
        dd(i) = dd(i) - w * dd(i-1);
    end

    % Обратный ход прогонки
    U = zeros(n,1);
    U(n) = dd(n) / bb(n);
    for i = n-1:-1:1
        U(i) = (dd(i) - cc(i) * U(i+1)) / bb(i);
    end

    % Оценка ошибки по Рунге
    if ~first
        U_fine_on_coarse_nodes = U(1:2:end);  % Берем каждую вторую точку
        err_est = max(abs(U_fine_on_coarse_nodes - U_prev)) / 3;
        fprintf('N = %d, h = %.6f, err_est = %.6e\n', N, h, err_est);

        if err_est < tol  % Достигнута нужная точность
            x_final = x;
            U_final = U;
            U_prev_for_plot = U_prev;
            x_prev_for_plot = x_prev;
            break;
        end
    else
        first = false;  % Первый шаг - просто сохраняем решение
    end

    % Удваиваем число узлов для следующей итерации
    U_prev = U;
    x_prev = x;
    N = 2*N - 1;
end

fprintf('Runge error estimate: %.6e\n', err_est);

% График численного и точного решения
figure;
plot(x_final, U_final, 'ro-', 'MarkerSize', 8, 'LineWidth', 1.5);
hold on;
plot(x_final, u_exact(x_final), 'b-', 'LineWidth', 1.5);
hold off;
xlabel('x'); ylabel('u(x)');
legend('Numerical', 'Exact');
grid on;

% График оценки ошибки по Рунге
figure;
x_coarse = x_prev_for_plot;
U_coarse = U_prev_for_plot;
U_fine_at_coarse_nodes = U_final(1:2:end);

error_estimate = abs(U_fine_at_coarse_nodes - U_coarse) / 3;
plot(x_coarse, error_estimate, 'ko-', 'MarkerSize', 8, 'LineWidth', 1.5);
xlabel('x'); ylabel('Error estimate (Runge)');
grid on;
\end{verbatim}

        k_minus_half = k(a - h/2);

        a_diag(N-1) = -(k_minus_half + k_plus_half)/h^2;
        b_diag(N)   = (k_minus_half + k_plus_half)/h^2 + qv(N);

        cc = c_diag(1:N-1);
        n = length(d_vec);
        bb = b_diag;
        dd = d_vec;

        for i = 2:n
            w = aa(i-1) / bb(i-1);
            bb(i) = bb(i) - w * cc(i-1);
            dd(i) = dd(i) - w * dd(i-1);
