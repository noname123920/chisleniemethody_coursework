\section*{Введение}
\addcontentsline{toc}{section}{Введение}

Многие физические, технические и экономические процессы описываются дифференциальными уравнениями. Особый класс составляют \textbf{двухточечные краевые задачи} для обыкновенных дифференциальных уравнений (ОДУ) второго порядка. Такие задачи возникают, когда известны условия не в одной точке, а на двух концах некоторого отрезка — отсюда название «двухточечные».

Математическая постановка задачи, рассматриваемой в данной работе, имеет вид:
\begin{equation*}
    -\bigl(k(x)\,u'\bigr)' + q(x)\,u = f(x), \quad a < x < b,
\end{equation*}
где $u(x)$ — искомая функция, $k(x)$, $q(x)$, $f(x)$ — заданные непрерывные функции, причём $k(x) > 0$ на всём отрезке $[a, b]$. Для единственности решения на границах отрезка $[a, b]$ задаются дополнительные условия — первого, второго или третьего рода.

Аналитическое решение такой задачи удаётся найти лишь для простейших случаев. Для реальных задач с переменными коэффициентами приходится использовать \textbf{численные методы}. Один из самых распространённых — \textbf{метод конечных разностей} (МКР). Суть метода заключается в замене непрерывной области $[a, b]$ набором отдельных точек (узлов), а производных — приближёнными формулами через значения функции в этих точках. В результате дифференциальное уравнение превращается в систему линейных алгебраических уравнений (СЛАУ), которую можно решить на компьютере.

\textbf{Цель работы:} программно реализовать метод конечных разностей для решения линейной двухточечной краевой задачи для ОДУ второго порядка с переменными коэффициентами.

Для достижения поставленной цели решаются следующие \textbf{задачи}:
\begin{enumerate}
    \item Описать постановку линейной двухточечной краевой задачи для ОДУ второго порядка с разными граничными условиями.
    \item Указать условия разрешимости краевой задачи.
    \item Изложить метод конечных разностей для решения этой задачи.
    \item Реализовать в среде Octave алгоритм решения краевой задачи методом конечных разностей для одномерного стационарного уравнения теплопроводности
    \begin{equation*}
        -\frac{d}{dx}\left(k(x)\frac{du}{dx}\right) + q(x)u = f(x), \quad a < x < b,
    \end{equation*}
    где $k(x) \neq \text{const}$, $q(x) \neq \text{const}$, $f(x) \neq \text{const}$ — непрерывные функции. Требуемая точность — $10^{-4}$.
    \item Оценить погрешность численного решения по правилу Рунге.
    \item Провести вычислительные эксперименты с различными краевыми условиями.
    \item Построить графики полученного решения и график погрешности.
\end{enumerate}

\newpage 
\section{Постановка краевой задачи с различными граничными условиями}
Рассматривается линейная двухточечная краевая задача для ОДУ второго порядка:
\begin{equation}
    -\frac{d}{dx}\left(k(x)\frac{du}{dx}\right) + q(x)u = f(x), \quad a < x < b,
    \label{eq:ode_main}
\end{equation}
где $u(x)$ --- искомая функция, $k(x)$, $q(x)$, $f(x)$ --- заданные непрерывные на $[a,b]$ функции, причём $k(x) > 0$.

Для однозначного определения решения на концах отрезка задаются граничные условия трёх возможных типов:

\begin{itemize}
    \item \textbf{Первый род (Дирихле):} фиксированы значения функции
    \begin{equation}
        u(a) = A, \quad u(b) = B.
    \end{equation}
    \item \textbf{Второй род (Неймана):} фиксированы значения производной
    \begin{equation}
        u'(a) = \gamma, \quad u'(b) = \delta.
    \end{equation}
    \item \textbf{Третий род (Робина):} задана линейная комбинация функции и производной
    \begin{equation}
        \alpha_0 u(a) + \alpha_1 u'(a) = \gamma_a, \quad \beta_0 u(b) + \beta_1 u'(b) = \gamma_b,
    \end{equation}
    где $\alpha_1 \neq 0$, $\beta_1 \neq 0$.
\end{itemize}

В дальнейшем для всех трёх типов граничных условий применяется одинаковый алгоритм дискретизации методом конечных разностей.

\newpage

\section{Условия разрешимости и единственности}
Для краевой задачи \eqref{eq:ode_main} справедливы следующие достаточные условия корректности:
\begin{enumerate}
    \item $k(x), q(x), f(x) \in C[a,b]$, $k(x) > 0$ на $[a,b]$.
    \item $q(x) \ge 0$ на $[a,b]$.
    \item Если $q(x) > 0$ хотя бы в одной точке отрезка $[a,b]$, задача имеет единственное классическое решение при любых граничных условиях (Дирихле, Неймана или Робина).
\end{enumerate}

В вычислительных экспериментах данной работы во всех тестовых примерах $q(x) > 0$ на $[0,1]$, что гарантирует существование и единственность решения для всех трёх типов граничных условий.

\newpage

\section{Вывод разностных формул, реализованных в программе}
В данном разделе показан полный вывод коэффициентов матрицы СЛАУ, получаемых при дискретизации краевой задачи методом конечных разностей. Для аппроксимации граничных условий используется метод фиктивных (мнимых) узлов \textit{(ghost nodes)}. Все выкладки приведены в соответствии с реализованным кодом Octave.

\subsection{Постановка задачи и введение сетки}
Рассматривается линейная двухточечная краевая задача:
\begin{equation}
    -\bigl(k(x)\,u'(x)\bigr)' + q(x)\,u(x) = f(x), \quad a < x < b,
    \label{eq:cont}
\end{equation}
с граничными условиями первого, второго или третьего рода. Функции \(k(x)>0\), \(q(x)\ge 0\), \(f(x)\) непрерывны на \([a,b]\).

Вводится равномерная сетка:
\begin{equation}
    x_i = a + (i-1)h, \quad i=1,2,\dots,N, \qquad h = \frac{b-a}{N-1}.
\end{equation}
Обозначения: \(u_i = u(x_i)\), \(q_i = q(x_i)\), \(f_i = f(x_i)\). Значения \(k\) в полуцелых узлах:
\begin{equation}
    k_{i+1/2} = k(x_i + h/2), \qquad k_{i-1/2} = k(x_i - h/2).
\end{equation}

\subsection{Внутренние узлы (\texorpdfstring{$i=2,\dots,N-1$}{i=2,...,N-1})}
Запишем уравнение \eqref{eq:cont} в узле \(x_i\):
\[
-\bigl(k u'\bigr)'(x_i) + q_i u_i = f_i.
\]
Производную потока аппроксимируем центральной разностью:
\[
\bigl(k u'\bigr)'(x_i) \approx \frac{ (k u')_{i+1/2} - (k u')_{i-1/2} }{h}.
\]
Значения \(u'\) в полуцелых узлах заменяем разностями вперёд/назад:
\[
u'_{i+1/2} \approx \frac{u_{i+1}-u_i}{h}, \qquad
u'_{i-1/2} \approx \frac{u_i-u_{i-1}}{h}.
\]
Тогда:
\[
(k u')_{i+1/2} \approx k_{i+1/2}\frac{u_{i+1}-u_i}{h}, \qquad
(k u')_{i-1/2} \approx k_{i-1/2}\frac{u_i-u_{i-1}}{h}.
\]
Подставляем в разностное уравнение:
\[
-\frac{ k_{i+1/2}\dfrac{u_{i+1}-u_i}{h} - k_{i-1/2}\dfrac{u_i-u_{i-1}}{h} }{h} + q_i u_i = f_i.
\]
После приведения к общему знаменателю:
\[
-\frac{ k_{i+1/2}(u_{i+1}-u_i) - k_{i-1/2}(u_i-u_{i-1}) }{h^2} + q_i u_i = f_i.
\]
Раскрываем скобки и группируем коэффициенты:
\begin{align}
&-\frac{k_{i+1/2}u_{i+1} - k_{i+1/2}u_i - k_{i-1/2}u_i + k_{i-1/2}u_{i-1}}{h^2} + q_i u_i = f_i,\nonumber\\
&-\frac{k_{i-1/2}}{h^2}u_{i-1} + \left(\frac{k_{i-1/2}+k_{i+1/2}}{h^2} + q_i\right)u_i - \frac{k_{i+1/2}}{h^2}u_{i+1} = f_i.
\label{eq:int}
\end{align}
Формула \eqref{eq:int} — основное разностное уравнение для внутренних узлов. В программной реализации оно выглядит следующим образом (цикл по \(i=2,\dots,N-1\)):

\begin{lstlisting}[caption={Фрагмент кода: внутренние узлы},label={lst:inner}]
for i = 2:N-1
    a_diag(i-1) = -kmid(i-1)/h^2;          % k_{i-1/2}
    c_diag(i)   = -kmid(i)/h^2;            % k_{i+1/2}
    b_diag(i)   = (kmid(i-1)+kmid(i))/h^2 + qv(i);
    d_vec(i)    = fv(i);
end
\end{lstlisting}

\subsection{Левая граница (\texorpdfstring{$x=a$}{x=a}, \texorpdfstring{$i=1$}{i=1})}
Вводим фиктивный узел \(x_0 = a-h\) со значением \(u_0\). Уравнение для \(i=1\) формально совпадает с \eqref{eq:int}, но содержит \(u_0\), которое выражается через граничное условие.

\subsubsection{Условие Неймана: \texorpdfstring{$u'(a)=\gamma$}{u'(a)=gamma}}
Центральная разность для производной:
\[
u'(a) \approx \frac{u_2-u_0}{2h} = \gamma \quad\Rightarrow\quad u_0 = u_2 - 2h\gamma.
\]
Подставляем в уравнение для \(i=1\):
\[
-\frac{ k_{1/2}(u_2-u_1) - k_{-1/2}\bigl(u_1 - (u_2-2h\gamma)\bigr) }{h^2} + q_1 u_1 = f_1.
\]
Упрощаем числитель:
\[
k_{1/2}(u_2-u_1) - k_{-1/2}(u_1-u_2+2h\gamma) = (k_{1/2}+k_{-1/2})u_2 - (k_{1/2}+k_{-1/2})u_1 - 2h k_{-1/2}\gamma.
\]
После подстановки и переноса слагаемого с \(\gamma\) в правую часть:
\[
\frac{k_{-1/2}+k_{1/2}}{h^2}u_1 - \frac{k_{-1/2}+k_{1/2}}{h^2}u_2 + q_1 u_1 = f_1 + \frac{2k_{-1/2}}{h}\gamma.
\]
Окончательно:
\begin{equation}
\boxed{\left(\frac{k_{-1/2}+k_{1/2}}{h^2} + q_1\right)u_1 - \frac{k_{-1/2}+k_{1/2}}{h^2}u_2 = f_1 + \frac{2k_{-1/2}}{h}\gamma.}
\label{eq:neumann_left}
\end{equation}
В коде это реализовано так:

\begin{lstlisting}[caption={Фрагмент кода: левая граница (Нейман)},label={lst:neumann_left}]
k_minus_half = k(a - h/2);
k_plus_half  = k(a + h/2);
b_diag(1) = (k_minus_half + k_plus_half)/h^2 + qv(1);
c_diag(1) = -(k_minus_half + k_plus_half)/h^2;
d_vec(1)  = fv(1) + 2*k_minus_half/h * gamma;
\end{lstlisting}

\subsubsection{Условие Робена: \texorpdfstring{$\alpha_0 u(a)+\alpha_1 u'(a)=\gamma_a$}{alpha0 u(a)+alpha1 u'(a)=gamma_a}}
Аппроксимация производной:
\[
\alpha_0 u_1 + \alpha_1\frac{u_2-u_0}{2h} = \gamma_a \quad\Rightarrow\quad u_0 = u_2 - \frac{2h}{\alpha_1}(\gamma_a - \alpha_0 u_1).
\]
Подставляем в разностное уравнение для \(i=1\):
\[
-\frac{ k_{1/2}(u_2-u_1) - k_{-1/2}\left(u_1 - u_2 + \frac{2h}{\alpha_1}(\gamma_a - \alpha_0 u_1)\right) }{h^2} + q_1 u_1 = f_1.
\]
После раскрытия скобок и приведения подобных получаем:
\begin{equation}
\boxed{\left(\frac{k_{-1/2}+k_{1/2}}{h^2} + q_1 + \frac{2k_{-1/2}\alpha_0}{h\alpha_1}\right)u_1 - \frac{k_{-1/2}+k_{1/2}}{h^2}u_2 = f_1 + \frac{2k_{-1/2}}{h\alpha_1}\gamma_a.}
\label{eq:robin_left}
\end{equation}
Реализация в коде:

\begin{lstlisting}[caption={Фрагмент кода: левая граница (Робен)},label={lst:robin_left}]
k_minus_half = k(a - h/2);
k_plus_half  = k(a + h/2);
b_diag(1) = (k_minus_half + k_plus_half)/h^2 + qv(1) + 2*k_minus_half*alpha0/(h*alpha1);
c_diag(1) = -(k_minus_half + k_plus_half)/h^2;
d_vec(1)  = fv(1) + 2*k_minus_half/(h*alpha1)*gamma_a;
\end{lstlisting}

\subsubsection{Условие Дирихле: \texorpdfstring{$u(a)=A$}{u(a)=A}}
В этом случае первое уравнение заменяется на тождество:
\[
u_1 = A,
\]
что в матрице даёт \(b_{11}=1\), \(c_{11}=0\), \(d_1=A\), а все остальные коэффициенты в первой строке обнуляются.

\begin{lstlisting}[caption={Фрагмент кода: левая граница (Дирихле)},label={lst:dirichlet_left}]
b_diag(1) = 1; c_diag(1) = 0; d_vec(1) = u_exact(a);
\end{lstlisting}

\subsection{Правая граница (\texorpdfstring{$x=b$}{x=b}, \texorpdfstring{$i=N$}{i=N})}
Вводим фиктивный узел \(x_{N+1}=b+h\) со значением \(u_{N+1}\).

\subsubsection{Условие Неймана: \texorpdfstring{$u'(b)=\delta$}{u'(b)=delta}}
\[
u'(b) \approx \frac{u_{N+1}-u_{N-1}}{2h} = \delta \quad\Rightarrow\quad u_{N+1} = u_{N-1} + 2h\delta.
\]
Разностное уравнение для \(i=N\):
\[
-\frac{ k_{N+1/2}(u_{N+1}-u_N) - k_{N-1/2}(u_N-u_{N-1}) }{h^2} + q_N u_N = f_N.
\]
Подставляем \(u_{N+1}\):
\[
-\frac{ k_{N+1/2}(u_{N-1}+2h\delta-u_N) - k_{N-1/2}(u_N-u_{N-1}) }{h^2} + q_N u_N = f_N.
\]
После упрощений:
\begin{equation}
\boxed{-\frac{k_{N-1/2}+k_{N+1/2}}{h^2}u_{N-1} + \left(\frac{k_{N-1/2}+k_{N+1/2}}{h^2} + q_N\right)u_N = f_N + \frac{2k_{N+1/2}}{h}\delta.}
\label{eq:neumann_right}
\end{equation}
Код:

\begin{lstlisting}[caption={Фрагмент кода: правая граница (Нейман)},label={lst:neumann_right}]
k_minus_half = k(b - h/2);
k_plus_half  = k(b + h/2);
a_diag(N-1) = -(k_minus_half + k_plus_half)/h^2;
b_diag(N)   = (k_minus_half + k_plus_half)/h^2 + qv(N);
d_vec(N)    = fv(N) + 2*k_plus_half/h * delta;
\end{lstlisting}

\subsubsection{Условие Робена: \texorpdfstring{$\beta_0 u(b)+\beta_1 u'(b)=\gamma_b$}{beta0 u(b)+beta1 u'(b)=gamma_b}}
\[
\beta_0 u_N + \beta_1\frac{u_{N+1}-u_{N-1}}{2h} = \gamma_b \quad\Rightarrow\quad u_{N+1} = u_{N-1} + \frac{2h}{\beta_1}(\gamma_b - \beta_0 u_N).
\]
Подстановка в разностное уравнение для \(i=N\) даёт:
\begin{equation}
\boxed{-\frac{k_{N-1/2}+k_{N+1/2}}{h^2}u_{N-1} + \left(\frac{k_{N-1/2}+k_{N+1/2}}{h^2} + q_N + \frac{2k_{N+1/2}\beta_0}{h\beta_1}\right)u_N = f_N + \frac{2k_{N+1/2}}{h\beta_1}\gamma_b.}
\label{eq:robin_right}
\end{equation}
Реализация:

\begin{lstlisting}[caption={Фрагмент кода: правая граница (Робен)},label={lst:robin_right}]
k_minus_half = k(b - h/2);
k_plus_half  = k(b + h/2);
a_diag(N-1) = -(k_minus_half + k_plus_half)/h^2;
b_diag(N)   = (k_minus_half + k_plus_half)/h^2 + qv(N) + 2*k_plus_half*beta0/(h*beta1);
d_vec(N)    = fv(N) + 2*k_plus_half/(h*beta1)*gamma_b;
\end{lstlisting}

\subsubsection{Условие Дирихле: \texorpdfstring{$u(b)=B$}{u(b)=B}}
Последняя строка матрицы заменяется на:
\[
u_N = B,
\]
т.е. \(b_{NN}=1\), \(a_{N,N-1}=0\), \(d_N = B\).

\begin{lstlisting}[caption={Фрагмент кода: правая граница (Дирихле)},label={lst:dirichlet_right}]
a_diag(N-1) = 0; b_diag(N) = 1; d_vec(N) = u_exact(b);
\end{lstlisting}

\subsection{Итоговая трёхдиагональная система и метод прогонки}
Все полученные уравнения вместе образуют СЛАУ с трёхдиагональной матрицей:
\[
A_i u_{i-1} + B_i u_i + C_i u_{i+1} = D_i,\quad i=1,\dots,N,
\]
где коэффициенты \(A_i, B_i, C_i, D_i\) определяются по формулам выше в зависимости от типа граничных условий. Система решается методом прогонки (алгоритм Томаса), реализованным в коде:

\begin{lstlisting}[caption={Метод прогонки (решение СЛАУ)},label={lst:thomas}]
aa = a_diag(1:N-1); bb = b_diag; cc = c_diag(1:N-1); dd = d_vec;
for i = 2:N
    w = aa(i-1) / bb(i-1);
    bb(i) = bb(i) - w * cc(i-1);
    dd(i) = dd(i) - w * dd(i-1);
end
U(N) = dd(N) / bb(N);
for i = N-1:-1:1
    U(i) = (dd(i) - cc(i) * U(i+1)) / bb(i);
end
\end{lstlisting}

Таким образом, полученные разностные формулы полностью соответствуют программной реализации в Octave и обеспечивают второй порядок аппроксимации на равномерной сетке.

\section{Реализация метода конечных разностей для задачи с граничными условиями первого рода (Дирихле)}

\subsection{Программная реализация на Octave}

Ниже приведён код решения задачи Дирихле с автоматическим сгущением сетки до достижения точности $\varepsilon = 10^{-4}$.

\begin{lstlisting}[caption={Code No.1},label={lst:dirichlet}]
graphics_toolkit("gnuplot");

a = 0; b = 1;
k = @(x) x.^2 + 1;
q = @(x) x + 1;
f = @(x) x.^5 - 21*x.^4 + 23*x.^3 - 14*x.^2 + 9*x - 2;

u_exact = @(x) x.^4 - 2*x.^3 + x.^2 + 3*x;

tol = 1e-4;
N = 5;
err_est = tol + 1;
first = true;

while err_est > tol
    h = (b - a)/(N - 1);
    x = linspace(a, b, N)';
    xmid = (x(1:end-1) + x(2:end))/2;

    kmid = k(xmid);
    qv = q(x);
    fv = f(x);

    a_diag = zeros(N,1);
    b_diag = zeros(N,1);
    c_diag = zeros(N,1);
    d_vec = zeros(N,1);

    for i = 2:N-1
        a_diag(i-1) = -kmid(i-1)/h^2;
        c_diag(i)   = -kmid(i)/h^2;
        b_diag(i)   = (kmid(i-1) + kmid(i))/h^2 + qv(i);
        d_vec(i)    = fv(i);
    end

    b_diag(1) = 1; c_diag(1) = 0; d_vec(1) = u_exact(0);
    a_diag(N-1) = 0; b_diag(N) = 1; d_vec(N) = u_exact(1);

    n = length(d_vec);
    aa = a_diag(1:n-1);
    bb = b_diag;
    cc = c_diag(1:n-1);
    dd = d_vec;

    for i = 2:n
        w = aa(i-1) / bb(i-1);
        bb(i) = bb(i) - w * cc(i-1);
        dd(i) = dd(i) - w * dd(i-1);
    end

    U = zeros(n,1);
    U(n) = dd(n) / bb(n);
    for i = n-1:-1:1
        U(i) = (dd(i) - cc(i) * U(i+1)) / bb(i);
    end

    if ~first
        U_coarse = U(1:2:end);
        err_est = max(abs(U_coarse - U_prev)) / 3;
        fprintf('N = %d, h = %.6f, err_est = %.6e\n', N, h, err_est);
        if err_est < tol
            x_final = x;
            U_final = U;
            break;
        end
    else
        first = false;
    end

    U_prev = U;
    N = 2*N - 1;
end

fprintf('Runge error estimate: %.6e\n', err_est);

figure;
plot(x_final, U_final, 'ro-', 'MarkerSize', 8, 'LineWidth', 1.5);
xlabel('x'); ylabel('u(x)');
legend('Numerical solution', 'Location', 'northwest');
grid on;

figure;
plot(x_final(2:end-1), abs(U_final(2:end-1) - U_prev(2:2:end-1)), 'k-o');
xlabel('x'); ylabel('Error estimate (Runge)');
grid on;
\end{lstlisting}

graphics_toolkit("gnuplot");

a = 0; b = 1;
k = @(x) x.^2 + 1;

\section*{Заключение}
\addcontentsline{toc}{section}{Заключение}

В ходе выполнения курсовой работы была реализована и исследована численная схема метода конечных разностей для решения линейной двухточечной краевой задачи для обыкновенного дифференциального уравнения второго порядка с переменными коэффициентами. Работа охватила полный цикл: от теоретического обоснования и вывода разностных формул до программной реализации в среде Octave и анализа результатов.

Основные результаты работы можно сформулировать следующим образом:

\begin{enumerate}
    \item \textbf{Теоретический анализ.} Рассмотрена постановка краевой задачи с граничными условиями первого, второго и третьего рода. Приведены достаточные условия разрешимости и единственности решения ($k(x)>0$, $q(x)\ge 0$), которые гарантируют корректность задачи для всех проведённых вычислительных экспериментов.

    \item \textbf{Разработка разностной схемы.} Получены конечно-разностные аппроксимации для внутренних узлов сетки и для всех трёх типов граничных условий. Для аппроксимации производных в граничных условиях использован метод фиктивных узлов (ghost nodes), что позволило единообразно строить разностные уравнения на границах и сохранить второй порядок аппроксимации. Выведенные формулы полностью соответствуют программной реализации.

    \item \textbf{Программная реализация.} В среде Octave разработаны три программы, решающие краевую задачу с различными граничными условиями:
    \begin{itemize}
        \item задача Дирихле (граничные условия первого рода);
        \item задача Неймана (граничные условия второго рода);
        \item задача Робена (граничные условия третьего рода).
    \end{itemize}
    Во всех трёх случаях реализован алгоритм автоматического сгущения сетки с контролем точности по правилу Рунге. Требуемая точность $\varepsilon = 10^{-4}$ была достигнута для каждого типа граничных условий.

    \item \textbf{Численные эксперименты.} Проведённые расчёты показали:
    \begin{itemize}
        \item Сходимость метода — при увеличении числа узлов сетки оценка погрешности монотонно убывает (см. таблицы \ref{tab:convergence}, \ref{tab:convergence2}, \ref{tab:convergence3}).
        \item Порядок сходимости соответствует теоретическому $O(h^2)$, что подтверждается уменьшением ошибки примерно в 4 раза при удвоении числа узлов.
        \item Для задачи Неймана потребовалась более мелкая сетка ($N=129$ для достижения точности $10^{-4}$) по сравнению с задачами Дирихле ($N=65$) и Робена ($N=65$), что связано с особенностями аппроксимации граничных условий второго рода.
    \end{itemize}

    \item \textbf{Оценка погрешности.} Построены графики распределения оценки погрешности по узлам сетки (рисунки \ref{fig:error}, \ref{fig:error2}, \ref{fig:error3}). Во всех случаях максимальная погрешность наблюдается во внутренней части области, а не на границах, что характерно для данного типа уравнений при заданных коэффициентах.

    \item \textbf{Визуализация результатов.} Полученные численные решения представлены в виде графиков (рисунки \ref{fig:solution}, \ref{fig:solution2}, \ref{fig:solution3}).
\end{enumerate}
