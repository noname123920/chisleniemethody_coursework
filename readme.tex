\section*{Введение}
\addcontentsline{toc}{section}{Введение}

Многие физические, технические и экономические процессы описываются дифференциальными уравнениями. Особый класс составляют \textbf{двухточечные краевые задачи} для обыкновенных дифференциальных уравнений (ОДУ) (то есть уравнений, связывающих функцию и её производные, при этом функция зависит только от одной переменной) второго порядка. Такие задачи возникают, когда известны условия не в одной точке, а на двух концах некоторого отрезка — отсюда название «двухточечные».

Математическая постановка задачи, рассматриваемой в данной работе, имеет вид:

\begin{equation*}
    -\frac{d}{dx}\left(k(x)\frac{du}{dx}\right) + q(x)u = f(x), \quad a < x < b
\end{equation*}
или в эквивалентной форме:
\begin{equation*}
    -\bigl(k(x)\,u'\bigr)' + q(x)\,u = f(x), \quad a < x < b.
\end{equation*}

Здесь $u(x)$ — искомая функция, $k(x)$, $q(x)$, $f(x)$ — заданные функции координаты, причём $k(x) > 0$.

На границах отрезка $[a, b]$ задаются дополнительные условия — первого, второго или третьего рода.

Аналитическое решение такой задачи удаётся найти лишь для простейших случаев. Для реальных задач с переменными коэффициентами $k(x)$, $q(x)$, $f(x)$ приходится использовать \textbf{численные методы}. Один из самых распространённых — \textbf{метод конечных разностей} (МКР). Суть метода заключается в замене непрерывной области $[a, b]$ набором отдельных точек (узлов), а производных — приближёнными формулами через значения функции в этих точках. В результате дифференциальное уравнение превращается в систему линейных алгебраических уравнений, которую можно решить на компьютере.

\textbf{Цель работы:}
\begin{enumerate}
    \item Описать постановку линейной двухточечной краевой задачи для ОДУ второго порядка с разными граничными условиями.
    \item Указать условия разрешимости краевой задачи.
    \item Описать метод конечных разностей для решения этой задачи.
    \item Реализовать в среде Octave алгоритм решения краевой задачи методом конечных разностей для одномерного стационарного уравнения теплопроводности
    \begin{equation*}
        -\frac{d}{dx}\left(k(x)\frac{du}{dx}\right) + q(x)u = f(x), \quad a < x < b,
    \end{equation*}
    где $k(x) \neq \text{const}$, $q(x) \neq \text{const}$, $f(x) \neq \text{const}$ — непрерывные функции, вид которых выбирается самостоятельно. Требуемая точность — $10^{-4}$.
    \item Оценить погрешность численного решения по правилу Рунге.
    \item Провести вычислительные эксперименты с различными краевыми условиями.
    \item Построить графики полученного решения и график погрешности.
\end{enumerate}

\section*{Постановка краевой задачи}
\addcontentsline{toc}{section}{Постановка краевой задачи}

Рассмотрим линейную двухточечную краевую задачу для обыкновенного дифференциального уравнения второго порядка на отрезке $[a, b]$:

\begin{equation}
    -\frac{d}{dx}\left(k(x)\frac{du}{dx}\right) + q(x)u = f(x), \quad a < x < b,
\end{equation}
или в эквивалентной записи:
\begin{equation}
    -\bigl(k(x)\,u'\bigr)' + q(x)\,u = f(x), \quad a < x < b.
\end{equation}

Здесь $u(x)$ — искомая функция, $k(x)$, $q(x)$, $f(x)$ — заданные непрерывные функции, причём $k(x) > 0$ на всём отрезке $[a, b]$.

Для единственности решения к уравнению (1) необходимо добавить граничные условия на концах отрезка. В зависимости от физической постановки могут задаваться условия первого, второго или третьего рода.

\subsection*{Граничные условия первого рода (условия Дирихле)}

На границах отрезка задаются значения самой искомой функции:

\begin{equation}
    u(a) = \alpha, \qquad u(b) = \beta
\end{equation}

где $\alpha$, $\beta$ — заданные числа.

\subsection*{Граничные условия второго рода (условия Неймана)}

На границах отрезка задаются значения производной искомой функции:

\begin{equation}
    u'(a) = \gamma, \qquad u'(b) = \delta
\end{equation}

где $\gamma$, $\delta$ — заданные числа.

\subsection*{Граничные условия третьего рода (смешанные)}

На границах отрезка задаётся линейная комбинация функции и её производной:

\begin{equation}
    \alpha_0 u(a) + \alpha_1 u'(a) = \gamma_a, \qquad \beta_0 u(b) + \beta_1 u'(b) = \gamma_b
\end{equation}

где $\alpha_0, \alpha_1, \beta_0, \beta_1, \gamma_a, \gamma_b$ — заданные числа, причём $|\alpha_0| + |\alpha_1| \neq 0$ и $|\beta_0| + |\beta_1| \neq 0$.

\subsection*{Условия разрешимости}

Для существования и единственности решения краевой задачи (1) необходимо выполнение определённых условий.

\begin{itemize}
    \item Если $q(x) \geq 0$ на $[a, b]$ и хотя бы в одной точке $q(x) > 0$, то задача с граничными условиями первого или третьего рода имеет единственное решение.
    \item Условие $k(x) > 0$ гарантирует, что уравнение не вырождается.
\end{itemize}

\section*{Описание метода}
\addcontentsline{toc}{section}{Описание метода}

\subsection{Идея метода конечных разностей}

Метод конечных разностей (МКР) — это способ приближённого решения дифференциальных уравнений. Основная идея: непрерывная область (отрезок $[a, b]$) заменяется конечным набором точек — \textbf{сеткой}, а все производные в уравнении заменяются \textbf{разностными отношениями} — приближёнными формулами, использующими значения функции в соседних точках. В результате дифференциальное уравнение превращается в систему линейных алгебраических уравнений (СЛАУ) относительно значений искомой функции в узлах сетки.

\subsection{Построение сетки}

Разобьём отрезок $[a, b]$ на $n$ равных частей. Шаг сетки:

\begin{equation}
    h = \frac{b-a}{n}.
\end{equation}

Узлы сетки:

\begin{equation}
    x_i = a + i h, \qquad i = 0, 1, 2, \dots, n.
\end{equation}

Крайние узлы $x_0 = a$ и $x_n = b$ называются \textbf{граничными}, остальные ($i = 1, \dots, n-1$) — \textbf{внутренними}.

Введём обозначение для приближённого значения функции в узле:

\begin{equation}
    u_i \approx u(x_i), \qquad i = 0, 1, \dots, n.
\end{equation}

Также нам понадобятся \textbf{полуцелые} точки (середины отрезков между узлами):

\begin{equation}
    x_{i-1/2} = x_i - \frac{h}{2} = \frac{x_{i-1} + x_i}{2}, \qquad 
    x_{i+1/2} = x_i + \frac{h}{2} = \frac{x_i + x_{i+1}}{2}.
\end{equation}

Значения $k(x)$ в этих точках будем обозначать:

\begin{equation}
    k_{i-1/2} = k(x_{i-1/2}), \qquad k_{i+1/2} = k(x_{i+1/2}).
\end{equation}

\subsection{Интегро-интерполяционный метод (метод баланса)}

Для аппроксимации уравнения

\begin{equation*}
    -\bigl(k(x)\,u'\bigr)' + q(x)\,u = f(x)
\end{equation*}

выбран \textbf{интегро-интерполяционный метод (метод баланса)}. Его преимущество перед прямой заменой производных состоит в том, что он:
\begin{itemize}
    \item сохраняет физический смысл задачи (баланс потоков),
    \item не требует вычисления производной $k'(x)$,
    \item пригоден для разрывных коэффициентов $k(x)$.
\end{itemize}

Рассмотрим внутренний узел $x_i$ ($i = 1, \dots, n-1$). Построим вокруг него \textbf{ячейку} — отрезок $[x_{i-1/2}, x_{i+1/2}]$ и проинтегрируем исходное уравнение по этой ячейке:

\begin{equation}
    -\int_{x_{i-1/2}}^{x_{i+1/2}} \bigl(k(x)\,u'\bigr)' \,dx 
    + \int_{x_{i-1/2}}^{x_{i+1/2}} q(x)\,u(x) \,dx 
    = \int_{x_{i-1/2}}^{x_{i+1/2}} f(x) \,dx.
\end{equation}

Первый интеграл берётся точно (по формуле Ньютона—Лейбница):

\begin{equation}
    \int_{x_{i-1/2}}^{x_{i+1/2}} \bigl(k\,u'\bigr)' \,dx 
    = k(x_{i+1/2})\,u'(x_{i+1/2}) - k(x_{i-1/2})\,u'(x_{i-1/2}).
\end{equation}

Второй и третий интегралы приближаем по формуле средних (считая, что $q$ и $f$ почти постоянны на длине ячейки):

\begin{equation}
    \int_{x_{i-1/2}}^{x_{i+1/2}} q(x)\,u(x) \,dx \approx q(x_i)\,u(x_i) \cdot h = q_i u_i h,
\end{equation}

\begin{equation}
    \int_{x_{i-1/2}}^{x_{i+1/2}} f(x) \,dx \approx f(x_i) \cdot h = f_i h,
\end{equation}

где $q_i = q(x_i)$, $f_i = f(x_i)$.

Производные $u'$ в полуцелых точках аппроксимируем центральной разностью второго порядка точности:

\begin{equation}
    u'(x_{i+1/2}) \approx \frac{u_{i+1} - u_i}{h}, \qquad 
    u'(x_{i-1/2}) \approx \frac{u_i - u_{i-1}}{h}.
\end{equation}

Подставляем всё вместе:

\begin{equation}
    -\left( k_{i+1/2} \frac{u_{i+1} - u_i}{h} - k_{i-1/2} \frac{u_i - u_{i-1}}{h} \right) 
    + q_i u_i h = f_i h.
\end{equation}

Делим обе части на $h$ (поскольку $h > 0$):

\begin{equation}
    -\frac{1}{h}\left( k_{i+1/2} \frac{u_{i+1} - u_i}{h} - k_{i-1/2} \frac{u_i - u_{i-1}}{h} \right) 
    + q_i u_i = f_i.
\end{equation}

Окончательно, разностное уравнение для $i$-го внутреннего узла:

\begin{equation}
    \boxed{-\frac{k_{i-1/2}}{h^2} u_{i-1} 
           + \left( \frac{k_{i-1/2} + k_{i+1/2}}{h^2} + q_i \right) u_i 
           - \frac{k_{i+1/2}}{h^2} u_{i+1} = f_i.}
\end{equation}

Это уравнение связывает три соседних неизвестных: $u_{i-1}$, $u_i$, $u_{i+1}$. Поэтому оно называется \textbf{трёхточечным}.

Для удобства перепишем его в каноническом виде:

\begin{equation}
    A_i u_{i-1} + C_i u_i + B_i u_{i+1} = F_i, \qquad i = 1, \dots, n-1,
\end{equation}

где

\begin{equation}
    A_i = -\frac{k_{i-1/2}}{h^2}, \qquad 
    B_i = -\frac{k_{i+1/2}}{h^2},
\end{equation}

\begin{equation}
    C_i = \frac{k_{i-1/2} + k_{i+1/2}}{h^2} + q_i, \qquad 
    F_i = f_i.
\end{equation}

\subsection{Аппроксимация граничных условий}

Граничные условия разных типов аппроксимируются по-разному. Рассмотрим каждый случай.

\subsubsection{Условия первого рода (Дирихле)}

Заданы значения на границах:

\begin{equation}
    u(a) = \alpha, \qquad u(b) = \beta.
\end{equation}

Аппроксимация тривиальна:

\begin{equation}
    u_0 = \alpha, \qquad u_n = \beta.
\end{equation}

При этом разностные уравнения для внутренних узлов ($i = 1, \dots, n-1$) остаются без изменений. В первом уравнении ($i=1$) член $A_1 u_0$ содержит известное $u_0$ и переносится в правую часть. Аналогично в последнем ($i=n-1$) член $B_{n-1} u_n$ содержит известное $u_n$.

Таким образом, для задачи Дирихле система состоит из $(n-1)$ уравнения относительно $(n-1)$ неизвестных $u_1, \dots, u_{n-1}$. Матрица системы остаётся трёхдиагональной:

\begin{align}
    C_1 u_1 + B_1 u_2 &= F_1 - A_1 u_0, \\
    A_i u_{i-1} + C_i u_i + B_i u_{i+1} &= F_i, \qquad i = 2, \dots, n-2, \\
    A_{n-1} u_{n-2} + C_{n-1} u_{n-1} &= F_{n-1} - B_{n-1} u_n.
\end{align}

\subsubsection{Условия второго рода (Неймана)}

Заданы значения производных на границах:

\begin{equation}
    u'(a) = \gamma, \qquad u'(b) = \delta.
\end{equation}

Производную аппроксимируем \textbf{центральной разностью} (второй порядок точности). Однако для этого нужно ввести \textbf{фиктивные узлы} за пределами отрезка: $x_{-1} = a - h$ и $x_{n+1} = b + h$.

Условие на левой границе ($x_0 = a$):

\begin{equation}
    \frac{u_1 - u_{-1}}{2h} = \gamma \quad \Longrightarrow \quad u_{-1} = u_1 - 2h\gamma.
\end{equation}

Условие на правой границе ($x_n = b$):

\begin{equation}
    \frac{u_{n+1} - u_{n-1}}{2h} = \delta \quad \Longrightarrow \quad u_{n+1} = u_{n-1} + 2h\delta.
\end{equation}

Теперь записываем разностное уравнение для $i=0$ (левая граница) и $i=n$ (правая граница). Фиктивные значения $u_{-1}$ и $u_{n+1}$ исключаем с помощью полученных выражений.

Для $i=0$:
\begin{equation}
    -\frac{k_{-1/2}}{h^2} u_{-1} + \left( \frac{k_{-1/2} + k_{1/2}}{h^2} + q_0 \right) u_0 
    - \frac{k_{1/2}}{h^2} u_1 = f_0.
\end{equation}

Подставляем $u_{-1} = u_1 - 2h\gamma$:
\begin{equation}
    -\frac{k_{-1/2}}{h^2} (u_1 - 2h\gamma) + \left( \frac{k_{-1/2} + k_{1/2}}{h^2} + q_0 \right) u_0 
    - \frac{k_{1/2}}{h^2} u_1 = f_0.
\end{equation}

После приведения подобных получаем уравнение для $i=0$:
\begin{equation}
    \left( \frac{k_{-1/2} + k_{1/2}}{h^2} + q_0 \right) u_0 
    - \frac{k_{-1/2} + k_{1/2}}{h^2} u_1 
    = f_0 + \frac{2k_{-1/2}}{h}\gamma. \quad \text{(исправлен знак)}
\end{equation}

Аналогично для $i=n$ получаем:
\begin{equation}
    -\frac{k_{n-1/2} + k_{n+1/2}}{h^2} u_{n-1} 
    + \left( \frac{k_{n-1/2} + k_{n+1/2}}{h^2} + q_n \right) u_n 
    = f_n + \frac{2k_{n+1/2}}{h}\delta.
\end{equation}

\subsubsection{Условия третьего рода (смешанные)}

Заданы линейные комбинации функции и её производной:

\begin{equation}
    \alpha_0 u(a) + \alpha_1 u'(a) = \gamma_a, \qquad 
    \beta_0 u(b) + \beta_1 u'(b) = \gamma_b.
\end{equation}

Вновь вводим фиктивные узлы $x_{-1}$ и $x_{n+1}$. Производные аппроксимируем центральными разностями:

\begin{equation}
    u'(a) \approx \frac{u_1 - u_{-1}}{2h}, \qquad 
    u'(b) \approx \frac{u_{n+1} - u_{n-1}}{2h}.
\end{equation}

Подставляем в граничные условия:

\begin{equation}
    \alpha_0 u_0 + \alpha_1 \frac{u_1 - u_{-1}}{2h} = \gamma_a,
\end{equation}
\begin{equation}
    \beta_0 u_n + \beta_1 \frac{u_{n+1} - u_{n-1}}{2h} = \gamma_b.
\end{equation}

Выражаем $u_{-1}$ и $u_{n+1}$:

\begin{equation}
    u_{-1} = u_1 + \frac{2h}{\alpha_1}(\alpha_0 u_0 - \gamma_a) \quad (\text{при } \alpha_1 \neq 0),
\end{equation}
\begin{equation}
    u_{n+1} = u_{n-1} - \frac{2h}{\beta_1}(\beta_0 u_n - \gamma_b) \quad (\text{при } \beta_1 \neq 0).
\end{equation}

Подставляем эти выражения в разностные уравнения для $i=0$ и $i=n$ и получаем замкнутую систему из $(n+1)$ уравнений с $(n+1)$ неизвестными.

Если $\alpha_1 = 0$ (или $\beta_1 = 0$), то условие вырождается в условие первого рода, которое рассматривается тривиально.

\subsection{Формирование СЛАУ}

После записи разностных уравнений для всех узлов получаем систему уравнений с трёхдиагональной матрицей. В матричном виде:

\begin{equation}
    \begin{pmatrix}
        C_0 & B_0 & 0 & \dots & 0 \\
        A_1 & C_1 & B_1 & \dots & 0 \\
        0 & A_2 & C_2 & \ddots & \vdots \\
        \vdots & \ddots & \ddots & \ddots & B_{N-1} \\
        0 & \dots & 0 & A_N & C_N
    \end{pmatrix}
    \begin{pmatrix}
        u_0 \\ u_1 \\ u_2 \\ \vdots \\ u_N
    \end{pmatrix}
    =
    \begin{pmatrix}
        F_0 \\ F_1 \\ F_2 \\ \vdots \\ F_N
    \end{pmatrix},
\end{equation}

где $N = n$ для условий Неймана и смешанных, и $N = n-1$ для условий Дирихле (при исключённых граничных узлах). В случае Дирихле можно также сохранить размерность $(n+1)$, положив $C_0 = 1$, $B_0 = 0$, $F_0 = \alpha$ и $A_n = 0$, $C_n = 1$, $F_n = \beta$.

\subsection{Метод прогонки}

Система с трёхдиагональной матрицей решается методом прогонки (алгоритм Томаса). Решение ищется в виде рекуррентного соотношения:

\begin{equation}
    u_i = P_i u_{i+1} + Q_i, \qquad i = 0, 1, \dots, N-1.
\end{equation}

\textbf{Прямой ход прогонки} — вычисляем прогоночные коэффициенты $P_i$ и $Q_i$. Из первого уравнения системы находим:

\begin{equation}
    P_0 = -\frac{B_0}{C_0}, \qquad Q_0 = \frac{F_0}{C_0}.
\end{equation}

Далее для $i = 1, \dots, N-1$:

\begin{equation}
    P_i = \frac{-B_i}{C_i + A_i P_{i-1}}, \qquad 
    Q_i = \frac{F_i - A_i Q_{i-1}}{C_i + A_i P_{i-1}}.
\end{equation}

\textbf{Обратный ход прогонки} — вычисляем $u_N$ из последнего уравнения:

\begin{equation}
    u_N = \frac{F_N - A_N Q_{N-1}}{C_N + A_N P_{N-1}}.
\end{equation}

Затем по формуле $u_i = P_i u_{i+1} + Q_i$ находим остальные значения от $i = N-1$ до $i = 0$.

Условие устойчивости прогонки (диагональное преобладание):
\begin{equation}
    |C_i| \geq |A_i| + |B_i| \quad \text{для всех } i.
\end{equation}
При $k(x) > 0$ и $q(x) \geq 0$ это условие гарантированно выполняется.

\subsection{Сходимость и точность}

Построенная разностная схема имеет \textbf{второй порядок аппроксимации} по $h$, то есть погрешность аппроксимации на одном шаге пропорциональна $h^2$. Для метода конечных разностей при достаточной гладкости решения и коэффициентов это означает, что

\begin{equation}
    \max_i |u(x_i) - u_i| \leq C \cdot h^2,
\end{equation}

где $C$ — некоторая постоянная, не зависящая от $h$. Таким образом, при уменьшении шага вдвое погрешность должна уменьшиться примерно в 4 раза.

\subsection{Оценка погрешности по правилу Рунге}

Для практической оценки погрешности используется \textbf{правило Рунге} (двойной пересчёт). Выполняем расчёт на двух сетках: с шагом $h$ (решение $u_i^{(h)}$) и с шагом $h/2$ (решение $u_i^{(h/2)}$). Поскольку схема имеет второй порядок точности, погрешность на сетке с шагом $h$ оценивается как:

\begin{equation}
    \varepsilon_i^{(h)} \approx \frac{|u_i^{(h/2)} - u_i^{(h)}|}{3},
\end{equation}

где $u_i^{(h/2)}$ в узлах грубой сетки берутся через один узел мелкой сетки (совпадающие узлы).

Максимальная оценка погрешности:

\begin{equation}
    \varepsilon_{\max}^{(h)} = \max_i \frac{|u_i^{(h/2)} - u_i^{(h)}|}{3}.
\end{equation}

Расчёт прекращается, когда $\varepsilon_{\max}^{(h)} < 10^{-4}$ (заданная точность).

\section*{Алгоритм численного решения}
\addcontentsline{toc}{section}{Алгоритм численного решения}

Общая последовательность действий при решении краевой задачи методом конечных разностей с адаптивным выбором шага по правилу Рунге выглядит следующим образом.

\medskip
\noindent\textbf{Шаг 1. Ввод исходных данных.}
\begin{itemize}
    \item Задают отрезок интегрирования $[a, b]$.
    \item Определяют функции $k(x)$, $q(x)$, $f(x)$.
    \item Выбирают тип граничных условий и задают соответствующие константы:
    \begin{itemize}
        \item для условий Дирихле: $\alpha$, $\beta$;
        \item для условий Неймана: $\gamma$, $\delta$;
        \item для смешанных условий: $\alpha_0, \alpha_1, \gamma_a$, $\beta_0, \beta_1, \gamma_b$.
    \end{itemize}
    \item Задают начальное число узлов сетки $n_0$ (например, $n_0 = 10$) и требуемую точность $\varepsilon = 10^{-4}$.
\end{itemize}

\medskip
\noindent\textbf{Шаг 2. Построение сетки.}
Вычисляют шаг $h = (b-a)/n$, где $n$ — текущее число разбиений (начинают с $n = n_0$). Формируют массив узлов $x_i = a + i h$, $i = 0, \dots, n$.

\medskip
\noindent\textbf{Шаг 3. Вычисление коэффициентов разностной схемы.}
Для каждого внутреннего узла $i = 1, \dots, n-1$:
\begin{itemize}
    \item находят значения $k_{i-1/2} = k(x_{i-1/2})$, $k_{i+1/2} = k(x_{i+1/2})$, где $x_{i \pm 1/2} = x_i \pm h/2$;
    \item вычисляют коэффициенты трёхдиагональной матрицы и правой части:
    \begin{align*}
        A_i &= -\frac{k_{i-1/2}}{h^2}, \\
        B_i &= -\frac{k_{i+1/2}}{h^2}, \\
        C_i &= \frac{k_{i-1/2} + k_{i+1/2}}{h^2} + q(x_i), \\
        F_i &= f(x_i).
    \end{align*}
\end{itemize}

\medskip
\noindent\textbf{Шаг 4. Учёт граничных условий.}
В зависимости от типа условий модифицируют первое ($i=0$) и последнее ($i=n$) уравнения системы.
\begin{itemize}
    \item \textit{Условия Дирихле:} полагают $u_0 = \alpha$, $u_n = \beta$, а во внутренних уравнениях переносят известные члены в правую часть. Размерность системы становится $n-1$.
    \item \textit{Условия Неймана:} вводят фиктивные узлы $x_{-1}$, $x_{n+1}$. Используя аппроксимацию производной центральной разностью, получают уравнение для $i=0$:
    \[
        \left( \frac{k_{-1/2}+k_{1/2}}{h^2} + q_0 \right)u_0 - \frac{k_{-1/2}+k_{1/2}}{h^2}u_1 = f_0 + \frac{2k_{-1/2}}{h}\gamma,
    \]
    и аналогичное уравнение для $i=n$:
    \[
        -\frac{k_{n-1/2}+k_{n+1/2}}{h^2}u_{n-1} + \left( \frac{k_{n-1/2}+k_{n+1/2}}{h^2} + q_n \right)u_n = f_n + \frac{2k_{n+1/2}}{h}\delta.
    \]
    Размерность системы остаётся $n+1$.
    \item \textit{Смешанные условия:} аналогично вводят фиктивные узлы и получают замкнутую систему из $n+1$ уравнений после исключения $u_{-1}$ и $u_{n+1}$.
\end{itemize}

\medskip
\noindent\textbf{Шаг 5. Решение СЛАУ методом прогонки.}
Полученную трёхдиагональную систему решают методом прогонки (алгоритм Томаса). Вычисляют прогоночные коэффициенты $P_i$, $Q_i$ прямым ходом, затем обратным ходом находят все $u_i$.

\medskip
\noindent\textbf{Шаг 6. Двойной пересчёт (правило Рунге).}
\begin{itemize}
    \item Сохраняют полученное решение $u_i^{(h)}$ (на сетке с шагом $h$).
    \item Увеличивают число узлов вдвое: $n := 2n$, $h := h/2$.
    \item Повторяют шаги 2–5 для новой сетки, получают $u_i^{(h/2)}$.
    \item В совпадающих узлах (каждый второй узел мелкой сетки) вычисляют оценку погрешности по правилу Рунге:
    \[
        \varepsilon_i^{(h)} = \frac{|u_i^{(h/2)} - u_i^{(h)}|}{3}.
    \]
    \item Находят максимальную погрешность $\varepsilon_{\max} = \max_i \varepsilon_i^{(h)}$.
\end{itemize}

\medskip
\noindent\textbf{Шаг 7. Проверка точности и адаптация.}
\begin{itemize}
    \item Если $\varepsilon_{\max} < \varepsilon$ (заданная точность), то решение $u_i^{(h/2)}$ на мелкой сетке принимают за окончательный результат, и алгоритм завершается.
    \item Иначе присваивают $u_i^{(h)} := u_i^{(h/2)}$, снова удваивают число узлов ($n := 2n$) и возвращаются к шагу 2.
\end{itemize}

\medskip
\noindent\textbf{Шаг 8. Вывод результатов.}
Формируют таблицы значений $u(x_i)$, вычисляют (если известно точное решение) фактическую погрешность, строят графики решения и график распределения погрешности.

\section*{Вычислительный эксперимент}

Вычислительный эксперимент состоит из двух этапов: тестирования программы на задаче с известным аналитическим решением и решения основной задачи с различными типами граничных условий.

\subsection*{Тестирование программы}

Для проверки корректности работы программы используется задача с простыми постоянными коэффициентами, для которой легко выписать точное решение.

\textbf{Постановка тестовой задачи.}
Рассматривается отрезок $[a, b] = [0, 1]$. Коэффициенты уравнения:
\[
k(x) = 1, \qquad q(x) = 0.
\]
Правая часть выбирается так, чтобы точное решение имело вид
\[
u_{\text{exact}}(x) = \sin(\pi x) + x^2.
\]
Подставляя это решение в уравнение
\[
-(k(x)\,u')' + q(x)\,u = f(x),
\]
получаем
\[
f(x) = \pi^2 \sin(\pi x) - 2.
\]
Граничные условия --- первого рода (Дирихле):
\[
u(0) = 0, \qquad u(1) = 1.
\]

\[
E_{\max}(n) = \max_{0 \le i \le n} |u_i - u_{\text{exact}}(x_i)|.
\]

\subsection*{Основная задача}

Для вычислительных экспериментов сформулирована задача с переменными коэффициентами. Отрезок прежний: $[a, b] = [0, 1]$.

Выбраны следующие функции коэффициентов:
\[
k(x) = 2 + \sin(3x), \qquad q(x) = 1 + x^2.
\]
Очевидно, $k(x) > 0$ и $q(x) > 0$ на всём отрезке.

В качестве точного решения (для оценки погрешности) используется функция
\[
u_{\text{exact}}(x) = e^x \cos(2x).
\]
Тогда правая часть уравнения вычисляется аналитически:
\[
f(x) = -\bigl(k(x)\,u'_{\text{exact}}(x)\bigr)' + q(x)\,u_{\text{exact}}(x).
\]
Конкретное выражение для $f(x)$ громоздко, но легко вычисляется программно по заданным функциям; здесь оно не приводится ради краткости.

Далее решается одна и та же задача с тремя различными типами граничных условий. Для каждого типа условия получаются подстановкой $x = 0$ и $x = 1$ в точное решение и его производную
\[
u'_{\text{exact}}(x) = e^x\bigl(\cos(2x) - 2\sin(2x)\bigr).
\]
